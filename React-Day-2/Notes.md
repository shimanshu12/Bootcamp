Components are independent and reusable block that return a react element.
## React Features:
1. Virtual DOM
"React uses a lightweight copy of the real DOM to batch updates and minimize re-renders, making UI updates fast."
2. Single Page Application
"React loads one HTML page and dynamically rewrites content without full page reloads, giving a smooth app-like experience."
3. JSX and Babel 
"JSX lets us write HTML in JS, and Babel compiles it to regular React.createElement calls browsers can understand."
4. One Way Data Binding
"Data flows from parent to child only, making app state predictable and easier to debug compared to two-way binding."
5. Component Based
"UI is broken into reusable, isolated pieces called components that manage their own state and logic."

## One-Liner for ‘Why React?’
"React gives fast, declarative, component-based UIs with a huge ecosystem and virtual DOM for performance."

## Hooks
 Hooks are functions
 There are two rules to use hooks:
  1. Only call hooks at the top level.
  2. Only call hooks from the React functions.
