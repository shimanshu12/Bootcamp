# Local storage --- data is permanent util we delete it manually
ex-> we put items in a cart
    Like saving files on your laptop → stays even after restart

# Session storage --- data remain until sesion is alive
Data exists only for that browser tab/session
ex -> we fill any form on site like closing tabs
      Like notes in a classroom → gone when class ends (tab closed)

# Cookies --- Data is small + can have expiry time + used for login/auth
1. Session cookies 
   Gets deleted automatically when browser is closed
   Real-life example: Like a temporary entry pass → valid only while you are inside the building
2. Persistent cookies 
   Stored for a fixed time (expiry date) and stays after closing browser
   Real-life example: Like a gym membership card → works for some days (e.g., 7 days)
3. Third party cookies 
   Created by other websites (ads/trackers) to track user behavior
   Real-life example: Like someone else tracking your shopping in a mall (ads company)
